Crime Finder CLI API

During my internship at Securus Technologies Inc., I developed this CLI webscraper that contributes to the R&D software department. This project revolves around a webscraping spider using Python 3 and the Scrapy framework. The purpose of this web scraper is to gather crime information in a particular area of interest provided by a latitude and longitude coordinate. The webscraper then uses a crime map api called Spotcrime to gather crimes near the coordinates. The different types of crimes include:
  • Arrest
  • Arson
  • Assault
  • Burglary
  • Robbery
  • Shooting
  • Theft
  • Vandalism
  • Other

Spotcrime API

SpotCrime is a crime map api that takes in coordinates and shows crimes near the point of interest. Crime data is claimed to be gathered from local agencies such as the Houston Police Department or the University of Houston Police department.

Instructions

The Crime Spider can be ran from the CLI. To run the file, the program takes in two arguments, a latitude and longitude value. If insufficient coordinates are provided, then the spider will not run.
General syntax: “python crimespider.py ‘lat’ ‘lon’ ”
Sample syntax: “python crimespider.py 34.436546 27.654654”

Output

The output of the result is in json format. The following information is returned:
  • Latitude coordinate
  • Longitude coordinate
  • Offense type
  • Adress
  • Date
  • Time

Summary

The crime information provided by the webscraper could greatly benefit the creation of zones of interest for offenders. The data collected by this program was purposed to help determine which areas had high crime density. The data can also show what types of crimes are more common in particular areas. The web crawler can be designed to pipeline a process while crawling, meaning the json data can be store in a database or an elastic cluster using SQLite or elasticscrapy respectively. A database can be populated periodically using a scheduled scraper. This information could then be used to graph trends or can be used for even machine learning. One limitation of this webscraper is that it is using the Spotcrime map api's html to gather the information. If the Spotcrime api changes its source code, then the webcrawler will be ineffective.
